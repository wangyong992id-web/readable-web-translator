# Chrome Web Store Listing

## Name

Readable Web Translator

## Short description

Translate webpages with your own AI API key. Bilingual, full-page, selection, and optional image translation.

## Detailed description

Readable Web Translator is a bring-your-own-key webpage translation extension. Configure your own model API keys and translate webpages in bilingual, full-page, selection, or optional image translation mode.

Key features:

- Bilingual translation: adds translated text near the original content while preserving page readability.
- Full-page translation: replaces visible webpage text for immersive reading.
- Automatic selection translation: when enabled, selected text is translated automatically without an extra click.
- Optional image text translation: show an image translation button on content images for screenshots, tables, charts, news images, and social media images.
- Separate text and image model channels: use different providers, models, and API keys for text and image translation.
- Protected terms: keeps URLs, usernames, tickers, money amounts, code snippets, and custom terms unchanged.
- Multi-language support: English, Simplified Chinese, Traditional Chinese, Japanese, Korean, and auto detection.
- Translation styles: natural, finance/technology, and faithful-natural.
- Per-site switch: enable or disable the extension on the current site.

Privacy and security:

- API keys are stored locally with `chrome.storage.local`.
- Webpage content scripts do not receive your API keys.
- Translation content is sent only to the model provider you configure.
- The extension does not operate a separate backend server, sell data, or collect analytics.

You need your own API key from a supported model provider before using translation features.
