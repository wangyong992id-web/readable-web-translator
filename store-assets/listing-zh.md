# Chrome 商店文案

## 名称

Readable Web Translator

## 简短描述

自带 AI API Key 的网页翻译插件，支持双语、全文、划词和图片文字翻译。

## 详细描述

Readable Web Translator 是一款自带大模型 API Key 的网页翻译插件。你可以分别配置文本模型和图片模型，例如文本使用 DeepSeek 控制成本，图片使用 Kimi 或 OpenAI 视觉模型提升识别能力。

核心功能：

- 双语对照翻译：尽量保留原网页排版，在原文附近追加译文。
- 全文翻译：将网页可见文本替换为目标语言，适合沉浸阅读。
- 划词自动翻译：开启后选中文字即可自动翻译，无需再次点击。
- 图片文字翻译：开启后内容图片上显示“译图”按钮，适合新闻截图、表格、推文截图和长图。
- 文本/图片通道分离：可分别选择服务商、模型和 API Key，方便平衡成本和质量。
- 保护词：自动保护 URL、用户名、股票代码、金额、代码片段和自定义术语。
- 多语言：支持英语、简体中文、繁体中文、日语、韩语和自动识别。
- 翻译风格：提供自然准确、金融科技、忠实自然三种风格。
- 当前站点开关：可以对单个网站关闭或启用插件，减少打扰。

隐私与安全：

- API Key 保存在本地 `chrome.storage.local`。
- 网页内容脚本不会读取 API Key。
- 翻译内容只会发送到你选择并配置的模型服务商。
- 插件不运行单独服务器，不出售数据，不收集分析数据。

适合场景：

- 阅读英文网页、推文、博客、新闻和研报。
- 阅读市场评论、科技内容、投资相关长文。
- 需要低成本调用 DeepSeek / Kimi / OpenAI API 的用户。
- 希望图片翻译和网页翻译可以分开配置模型的用户。

使用前需要准备对应模型服务商的 API Key。
